# Pong

Requirements:
1) Git
2) Cmake

How to build on Windows:
1. git clone https://github.com/Faizan1015/Pong.git
2. mkdir Pong/build
3. cd Pong/build
4. cmake ..
5. cmake --build . --config Release

How to build on Mac:
1. git clone https://github.com/Faizan1015/Pong.git
2. mkdir Pong/build
3. cd Pong/build
4. cmake ..
5. make
